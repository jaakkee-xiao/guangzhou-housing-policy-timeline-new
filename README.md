# 广州房票政策时间线网站

这是一个展示广州房票政策发展历程的交互式时间线网站，包含两个AI模型的分析内容。

## 功能特性

- 📊 **双AI模型分析展示**：GPT模型分析 + 更新所平台AI分析
- 🔄 **内容切换功能**：点击按钮可在两个AI模型的分析内容间切换
- 📱 **响应式设计**：支持各种设备访问
- ✨ **美观的时间线界面**：展示广州房票政策发展历程
- 🎯 **平滑动画效果**：提供良好的用户体验

## 本地运行

1. 下载项目文件
2. 在项目目录中运行以下命令之一：

```bash
# 使用Python
python -m http.server 8080

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8080
```

3. 在浏览器中访问 `http://localhost:8080`
